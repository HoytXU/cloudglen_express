Cloudglen Express

A procedural landscape built from noise, layers, and masks

Press F for fullscreen

From one function to a complete scene

The image is generated without photographs, meshes, or a 3D camera. A smooth random function defines one cloud edge. Weighted copies add detail, repeated layers create depth, and simple masks draw the train.

1

A continuous random line

Value noise interpolates between random lattice values. It remains unpredictable at large scales while changing continuously between samples.

n(x) = lerp(ri, ri+1, s(fract(x)))

float f = fract(x);
f = f*f*f*(f*(f*6.0-15.0)+10.0);
float y = mix(a, b, f);

2

One cloud boundary

Several noise scales are added together. Large features carry the silhouette; faster and weaker signals supply progressively finer detail.

C(x) = ΣK−1k=0 akn(2kx)ΣK−1k=0 ak 

float sum = 0.0;
for (int k = 0; k < K; ++k) {
  sum += amplitude * noise(p);
  p *= 2.0;
  amplitude *= 0.7;
}

3

Layers and parallax

The same cloud function is evaluated at several heights and phases. Distant layers move slowly; nearby layers move quickly and cover the layers behind them.

hi(x,t) = mi + Ai [C(x + t/di + φi) − ½]

m: height  ·  A: variation
d: distance  ·  φ: phase

4

Train and bridge masks

Rectangles define the train body, distance tests define the wheels, and repeated coordinates define the bridge. Their hard silhouette remains legible against the continuous cloud boundaries.

Mtrain = max(Mbody, Mroof, Mwheels)

float wheel = 1.0 - step(
  radius, length(uv - center)
);
color = mix(color, trainColor, mask);

5

Final expression

Each pixel is formed by drawing the distant layers first, then the bridge and train, and finally the foreground clouds and smoke.

F(p,t) = V(p) [Lbg ⊕ Mbridge ⊕ Mtrain ⊕ Lfg ⊕ Ssmoke]

The operator ⊕ means that each term is drawn over the terms before it.
